# GRN-REIN Project

Algorithms for Synthesis of Gene Regulatory Networks using RE:IN.

## Quick Start
See `VM_ACCESS_GUIDE.md` to connect to the engine VM.
See `GRN_EXPERIMENT_PLAN.md` for the full experiment strategy.

## Repository Structure
- `models/` — source .aeon files organized by experiment
- `pipeline/` — Python scripts for conversion and analysis
- `results/` — RE:IN engine output JSONs
- `ground_truth/` — ground truth CSVs from the GT pipeline
- `reports/` — final reports and summary CSVs
